# Overview

This directory contains examples referenced by the Cortex tutorials. See that documentation for
details.

# Quickstart

Try the following (a sampling):
```
    # Terminal 1:
    ./cortex launch

    # Terminal 2:
    ./cortex activate franka/open_gripper.py
    ./cortex activate franka/close_gripper.py
    ./cortex activate franka/nullspace.py
    ./cortex activate franka/build_block_tower.py
```
