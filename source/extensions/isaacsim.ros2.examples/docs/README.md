# ROS 2 Examples Extension

This extension provides ROS 2 example scenes and demonstrations for Isaac Sim.

## Overview

The `isaacsim.ros2.examples` extension includes several ready-to-use examples that demonstrate ROS 2 integration with Isaac Sim:

### MoveIt Integration Example
- **Extension**: `isaacsim.ros2.examples.ros_moveit_sample`
- Demonstrates how to integrate MoveIt motion planning with Isaac Sim
- Includes a Franka Panda robot setup with pre-configured OmniGraph nodes

### Navigation Examples
- **Extension**: `isaacsim.ros2.examples.ros_samples`
- Multiple navigation scenarios including:
  - Nova Carter warehouse navigation
  - Nova Carter with joint states
  - iw_hub warehouse navigation
  - Isaac ROS Perceptor scene
  - Multi-robot hospital navigation
  - Multi-robot office navigation

### Waypoint Follower Example
- **Extension**: `isaacsim.ros2.examples.ros_waypoint_follower_sample`
- Demonstrates waypoint following and robot patrolling
- Includes both single waypoint and multi-waypoint patrolling modes
- Interactive UI for configuring waypoint parameters

## Usage

Each example is registered with the Isaac Sim Examples Browser and can be accessed through the main menu under their respective categories:
- ROS2/MoveIt
- ROS2/Navigation
- ROS2/Isaac ROS
