Isaac Sim Asset Transformer
===========================

Purpose
-------
Python-only extension that provides utilities for transforming assets within Isaac Sim (e.g., batch transforms, renaming, metadata operations). Initial version exposes the extension scaffold and lifecycle logging only.

Usage
-----
Enable `isaacsim.asset.transformer` in the Extension Manager. Lifecycle events are logged to the console.

Development
-----------
- Entry point: `isaacsim.asset.transformer.extension.Extension`.
- Version: 0.1.0.
