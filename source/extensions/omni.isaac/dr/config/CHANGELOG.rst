[0.1.0] - 2020-07-29
========================

Added
------
- Based on USD Schema
- Support randomization of:
    - Materials and its properties like color, texture etc.
    - Light parameters
    - Object position (supports look at)
    - Object orientation
    - Object scale
    - Object visibility
- Create component to enable various randomization behavior
    - Color component
    - Texture component
    - Material component
    - Light component
    - Movement component
    - Rotation component
    - Scale component
    - Visibility component
    - Mesh component
- Invoke randomization manually (based on user request) or given duration
- Specify seed to introduce consistency in the randomization process
- Documentation: https://isaac.gitlab-master-pages.nvidia.com/omni_isaac_sim/text/Robotics_DR.html
