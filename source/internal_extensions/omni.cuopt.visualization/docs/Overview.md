# Overview

This extension enables helper utilities to process and visualize waypoint graph data and routing solutions in Isaac Sim for use with cuOpt.


## Recommended Use
- This extension is intended to be used alongside omni.cuopt.service which requires either an access to a running instance of the cuOpt microservice (local or remote) or credentials to the cuOpt managed service. Install [cuOpt](https://github.com/NVIDIA/cuopt/) or experience and access managed service [here](https://build.nvidia.com/nvidia/nvidia-cuopt)


- The code for this extension is available and can be extended for specific use cases as needed.


- If using the cuOpt service, a complete list of features can be found in the [cuOpt docs](https://docs.nvidia.com/cuopt/)
