# Generic Model Output Python Package

**Generic Model Output** is published as a Python (PIP, wheel) package. This means that you can use **Generic Model Output** as a Python library in your Python projects.

```bash
pip install generic-model-output --extra-index-url https://urm.nvidia.com/artifactory/api/pypi/ct-omniverse-pypi
```
