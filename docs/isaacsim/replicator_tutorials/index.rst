========================================
Perception Data Generation (Replicator)
========================================


|isaac-sim_short| Replicator offers various tools and workflows for synthetic data generation (SDG), primarily using the :doc:`omni.replicator<extensions:ext_replicator>` extension. The examples and tutorials in this section showcase practical applications for robotics, including domain randomization, sensor simulation, and data collection with :doc:`annotators<extensions:ext_replicator/annotators_details>` and :doc:`writers<extensions:ext_replicator/writer_examples>`.
