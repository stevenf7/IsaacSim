# OAK-D ToF (Certified)

The [Luxonis OAK-D ToF](https://shop.luxonis.com/products/oak-d-tof) is an RVC2-generation depth camera with Time-of-Flight (3D) and color stereo cameras (OV9782). This model does not include a dedicated RGB camera, instead providing dual depth sensing through both stereo disparity and Time-of-Flight methods.
