=========================
Replicator Menu
=========================

The Replicator (Tools > Replicator) menu has a suite of useful tools and extensions for :doc:`omni.replicator<extensions:ext_replicator>` and generating, visualizing, and recording synthetic data.

=======================  ==========================================================
UI Element               Reference
=======================  ==========================================================
Semantics Schema Editor  Add semantic information. Refer to the :doc:`Semantics Schema Editor <extensions:ext_replicator/semantics_schema_editor>` docs for more details.
Synthetic Data Recorder  Synthetic Data Recorder. Refer to the :doc:`Synthetic Data Recorder </replicator_tutorials/tutorial_replicator_recorder>` docs for more details.
ReplicatorYAML           Generating dataset using :ref:`Replicator YAML<replicator_yaml_manual>`.
Start                    Starts the randomizations and writing to disk  :doc:`omni.replicator<extensions:ext_replicator>`.
Step                     Performs a single randomization operations with writing to disk  :doc:`omni.replicator<extensions:ext_replicator>`.
Preview                  Performs a singe randomization iteration without writing to disk :doc:`omni.replicator<extensions:ext_replicator>`.
=======================  ==========================================================
