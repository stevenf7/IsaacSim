.. _isaac_sim_create:

========================
Create Menu
========================

This table outlines the common things needed for scene creation and layout in Isaac Sim under the Create tab.


============================== =====================================================================
Menu Items                     Action
============================== =====================================================================
Light                          | Create custom Lights. See the Lighting docs for details.
Camera                         | Add Cameras to the stage.
Material                       | Create Omniverse Materials based on templates.
Physics                        | See the :doc:`Physics <kit-physics:index>` docs for details.
Sensors                        | Adds :ref:`cameras<isaacsim_sensors_camera>` and :ref:`RTX-based<isaacsim_sensors_rtx>`, :ref:`PhysX-based<isaacsim_sensors_physx>`, or :ref:`physics-based<isaacsim_sensors_physics>` sensors to the Stage.
Robots                         | Adds different robots to the Stage.
Environments                   | Adds different environments to the Stage.
April Tag                      | Adds :ref:`April Tag Asset <isaac_sim_asset_april_tag>` to the Stage.
============================== =====================================================================
