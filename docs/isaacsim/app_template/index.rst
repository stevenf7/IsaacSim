===================================
Application Template
===================================

.. deprecated:: 6.0.0
   Isaac Sim 5.1.0 was the final release of the standalone Application Template (``isaacsim-app-template``). Starting with Isaac Sim 6.0.0, its functionality is integrated into the open-source `Isaac Sim repository <https://github.com/isaac-sim/IsaacSim>`_.

Isaac Sim can also be installed and customized via the `application template repo  <https://github.com/isaac-sim/isaacsim-app-template>`_.
This repository allows you to build lightweight apps that pull extensions from the registry directly.
We provide a set of default applications that mirror the main Isaac Sim binary application configs as well as a set of templates that can be customized and extended for your own applications.

For more details and a quick start guide see `here <https://github.com/isaac-sim/isaacsim-app-template>`_.
