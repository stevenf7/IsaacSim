Azure Deployment
############################################

Requirements
---------------------------

The requirements for running |isaac-sim| on Microsoft Azure are:

* A Microsoft Azure Portal account that is able to launch a Virtual Machine with GPU support.

* The Remote Desktop Connection application for connecting into a Windows-based VMI.

* To connect to a graphical desktop on Linux-based VMIs, an application such as `Cendio's ThinLinc <https://www.cendio.com//>`__ is required to be installed on both the VDI (server) and local workstation (client).
