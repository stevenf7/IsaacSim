=================================
ROS 2 
=================================

`ROS (Robot Operating System) <https://www.ros.org/>`_ is a set of software libraries and tools for building robotics applications. Isaac Sim is connected to ROS through the ROS 2 bridge extension.
**We recommend ROS 2 Humble and Jazzy** with |isaac-sim|.


The following tutorials detail the process of extracting and manipulating synthetic data generated by |isaac-sim| and publishing them to your ROS applications.
